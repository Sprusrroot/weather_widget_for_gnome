# weather_widget_for_gnome
Weather Widget For Gnome

**INSTRUCTION:**
  1. Download this repo
  2. Install Hidamari
  3. Change City and API (openweathermap) in scipt.js file (also you can change wallpaper image in style.css file).
  4. Open Hidamari
  5. Select Web Page -> Local Web Page -> select index.html file

*im use this icons:* https://basmilius.github.io/weather-icons/index-fill.html
